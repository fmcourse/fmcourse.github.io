Важно!

1. При первом открытии окошка Install надо убрать
   галочку "Contact all update sites ...", иначе не получится
   установить все плагины.

2. После установки SMT Solvers надо вручную установить
   солверы. Для этого открыть Install, выбрать Rodin Plug-ins
   в Work with, поставить галочки напротив SMT CVC3, SMT CVC4,
   SMT veriT, SMT Z3 и нажать Next.
